# Project Name
FakeSafeAi:

## Problem Statement
A large number of startup ideas appear promising on the surface but fail due to hidden risks, unrealistic assumptions, and founder bias.
Early-stage founders often overestimate market demand, underestimate competition, and ignore psychological biases like overconfidence and confirmation bias.

From an investor’s perspective, these gaps make it difficult to trust early ideas. From a risk analyst’s perspective, many ideas lack structured evaluation. From a psychological perspective, human bias heavily influences decision-making.

There is a need for a system that evaluates startup ideas from multiple expert lenses simultaneously to reveal what founders might overlook.

## Project Description
Investor Lens
Evaluates ROI potential

Market size and scalability

Funding attractiveness

Exit possibilities

 2. Risk Analyst Lens
Identifies operational, financial, and market risks

Detects feasibility gaps

Highlights failure points

Assigns a structured risk score

 3. Psychological Lens
Detects founder bias (overconfidence, survivorship bias)
Multi-dimensional risk report

“Reality Check” insights

Risk score & credibility score

Actionable improvements
